# Bash Security Scripts

Scripts for network administration and security auditing on Linux (Kali/Ubuntu).
Bash scripts will be added once copied from the Kali environment.

## Scripts

- **network_info.sh** — Displays network interfaces, IPs, and active connections
- **quick_scan.sh** — Automated quick network scan using Nmap
- **network_compare.sh** — Compares two scans to detect new/removed devices
- **security_check.sh** — System security audit: users, ports, services, permissions
- **firewall_setup.sh** — Configures iptables firewall rules
