| Command | What it does |
|---------|-------------|
| cp | Copy files |
| mv | Move or rename files |

## Day 4 - Processes
| Command | What it does |
|---------|-------------|
| ps aux | Shows running processes |
| top | Live process monitor |
| kill | Stops a process |
| df -h | Shows disk space |
| free -h | Shows memory usage |

## Day 5 - Pipes and Redirection
| Symbol | What it does | Example |
|--------|-------------|---------|
| > | Sends output to file (overwrites) | ls > files.txt |
| >> | Sends output to file (appends) | echo "new" >> files.txt |
| pipe | Sends output to another command | cat file.txt pipe grep "error" |
